Server module
=================================================

Серверный модуль мессенджера.
Обрабатывает сообщения, регистрирует пользователей, авторизует их, хранит публичные ключи.

Использование:

Модуль на данный момент не поддерживает аргументы командной строки

Примеры использования:

``python server.py``

*Запуск сервера*

~~~~~~~~~
