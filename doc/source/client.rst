Client module
=================================================

Клиентское приложение для обмена сообщениями.
Поддерживает отправку сообщений пользователям в онлайн и прием сообщений от них.
Сообщения шифруются с помощью алгоритма RSA с длиной ключа 2048 bit.
Приватный ключ хранится только на стороне клиента

Поддерживает аргументы коммандной строки:

``python client.py -u {имя пользователя}``

-u - имя пользователя, с которым произойдёт вход в систему.

Имя пользователя является необязательным параметром.
Если оно введено - поле в интерфейсе будет предзаполнено значением.

Примеры использования:

* ``python client.py``

*Запуск приложения с параметрами по умолчанию.*

* ``python client.py -u test1``

*Запуск приложения с пользователем test1*

client.py
~~~~~~~~~

Запускаемый модуль, содержит парсер аргументов командной строки, функционал для инициализации приложения и
предзаполняет графический интерфейс входа в систему.

При запуске, появляется окно конфигурации входа в систему: имя пользователя, его пароль, адрес и порт сервера, к
которому необходимо подключиться.
